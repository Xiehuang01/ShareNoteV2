# 🎉 欢迎使用 ShareNote！

欢迎来到 **ShareNote** - 一个现代化的笔记分享平台！这里是你开始使用的第一步。

---

## 🚀 快速开始

### 什么是 ShareNote？

ShareNote 是一个功能强大的在线笔记管理和分享平台，让你可以：

- 📝 **创建和编辑** Markdown 笔记
- 🖼️ **上传和查看** 图片、PDF 等多种文件格式
- 📁 **组织管理** 你的所有笔记和文件
- 👥 **分享协作** 与团队成员共同编辑
- 🔍 **快速搜索** 找到你需要的内容

---

## 📖 新手指南

### 第一步：上传你的第一个笔记

1. 点击左侧面板顶部的 **"上传笔记"** 按钮
2. 选择你的 Markdown 文件（.md）或其他文档
3. 文件上传后会自动出现在笔记列表中

### 第二步：浏览和管理笔记

- **点击笔记** 即可在右侧查看内容
- **Markdown 文件** 会自动渲染，支持语法高亮
- **图片文件** 支持缩放和平移查看
- **PDF 文件** 可以直接在线预览

### 第三步：使用目录功能

对于 Markdown 文件，ShareNote 会自动生成目录：

- 点击右上角的 📋 **目录按钮** 展开/收起目录
- 点击目录中的标题可以快速跳转到对应章节
- 目录会根据你的标题层级自动缩进

---

## 🎯 核心功能

### 📝 Markdown 支持

ShareNote 完全支持 Markdown 语法，包括：

- **标题**：使用 `#` 创建不同级别的标题
- **代码块**：支持语法高亮
- **表格**：自动格式化表格显示
- **链接**：支持内部和外部链接
- **图片**：嵌入图片显示

### 🖼️ 多媒体文件

- **图片格式**：PNG, JPG, JPEG
- **文档格式**：PDF
- **交互功能**：图片缩放、平移

### 🔍 智能搜索

使用顶部搜索框快速找到你需要的笔记：
- 支持文件名搜索
- 支持内容关键词搜索

---

## 💡 使用技巧

### 快捷操作

- **快速预览**：点击笔记列表中的任意笔记即可预览
- **目录导航**：使用目录快速跳转到文档的任意部分
- **文件管理**：通过左侧面板管理所有上传的文件

### 最佳实践

1. **文件命名**：使用有意义的文件名，便于搜索和管理
2. **Markdown 结构**：合理使用标题层级，让目录更清晰
3. **定期整理**：及时删除不需要的文件，保持工作区整洁

---

## 🛠️ 界面介绍

### 左侧导航面板

- **折叠/展开按钮**：控制面板显示
- **首页按钮**：返回主页
- **设置按钮**：访问个人设置
- **用户信息**：显示当前用户信息
- **笔记列表**：显示所有上传的文件

### 右侧内容区域

- **标题栏**：显示当前文件名和操作按钮
- **内容区**：显示文件内容
- **目录面板**：Markdown 文件的导航目录

### 操作按钮

- 📋 **目录**：展开/收起目录面板
- 🗑️ **删除**：删除当前文件
- ✏️ **编辑**：编辑当前文件
- 💾 **保存**：保存修改
